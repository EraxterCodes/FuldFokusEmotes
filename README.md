# FuldFokusEmotes
This addon extends the [Twitch Emotes v2](https://www.curseforge.com/wow/addons/twitch-emotes-v2) with emotes used by the guild FuldFokus on EU-Draenor. 
Addon is maintained by [Eraxter](https://github.com/EraxterCodes). All credit goes to the original authors as this is injects addtional emotes.
